# Pokémon Webpage

## Description
Interactive Pokémon web application built with HTML, CSS, and JavaScript that fetches data from the PokéAPI to display Pokémon information across multiple generations.

## Technologies
- HTML
- CSS
- JavaScript
- PokéAPI

## Features
- Random Pokémon generator
- Dynamic Pokémon cards
- Name toggling and interactive UI elements
- Data fetched from Generation 1 through Generation 9
- Responsive layout and styling

## Overview
This project was created to practice front end web development concepts and working with external APIs. It focuses on building an interactive and user-friendly interface while dynamically rendering data from the PokéAPI.

## Notes
This project was developed as a personal and academic project to strengthen skills in JavaScript, API integration, and UI design.
